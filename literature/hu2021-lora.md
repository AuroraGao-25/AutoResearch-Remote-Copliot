# Hu et al. (2021) — LoRA

- **Source:** https://arxiv.org/abs/2106.09685
- **DOI:** 10.48550/arXiv.2106.09685
- **Key abstract takeaway:** Low-rank adaptation dramatically reduces trainable parameters while preserving downstream quality.
- **Why it matters:** Provides adaptation strategy for resource-constrained fine-tuning.
- **Use in this project:** Baseline PEFT framing and rationale for parameter-efficient targeted adaptation.
