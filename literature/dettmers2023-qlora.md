# Dettmers et al. (2023) — QLoRA

- **Source:** https://arxiv.org/abs/2305.14314
- **DOI:** 10.48550/arXiv.2305.14314
- **Key abstract takeaway:** 4-bit quantized finetuning through frozen base model with LoRA enables strong performance under tight memory budgets.
- **Why it matters:** Operationally critical for Colab T4 / limited-GPU ABSA experiments.
- **Use in this project:** Main adaptation method for A1/A2 and targeted augmentation ablations.
